Song Presets

Cypher 4 (BTS)
IDOL (BTS)
DNA (BTS)
Syntax: <bpm> <Part Letter> <Beats><Beats>... <Part Letter>...
# = Forward immediatly to next on button press
> = Jump back here after end
Part Letter is defined in Song part configuration
Beats are one digit numbers, that get accumulated: 8888=32 Beats
248 =14 Beats
Syntax: <Part Letter><Pattern>/<Speed>:<color>...
Pattern: identifies the pattern of light change (see pattern catalog)
Speed: Speed of the pattern 2=Haft Beat 4=Beat 8=8th, 16,32,64
Color: defines the color palette (see color palette catalog)

Pattern Catalog

Groups of same pattern varies in speed of color change relative to beat
0-3: Pulse
4-7: Heartbeat Pulse
8-9: Dimmed Pulse, Heartbeat
10: Whipe 11: Whipe over black
20-29: Orbit
30-39: Color Orbit, color shift 1
40-49: " , color shift 2
50-59: " , color shift 3
60-69: Full Rainbow
70-79: Half Rainbow
80-89: Quater Rainbow
90-99: Flat (Varies in distance to next color)

Color Palette Catalog

0..: Broad coloring
20..: Pink red orange yellow coloring
50..: Cyan Blue Purple coloring
50..: Green Lemon coloring
100..: Pulse optimized
110..: Disco shift 2 optimized
120..: doubled entries (whipe optimized)
